# Static Visual Preview

These are **static snapshots** of the ThreatNova CyberLabs UI, generated from the running
app so the design can be previewed in a browser without running anything.

**Open `index.html`** and click through the top navigation to tour the screens
(landing, courses, course detail, lesson, labs, lab challenge, dashboard, certificate,
verification, and the admin console).

> ⚠️ These pages are inert — login, enrollment, and flag submission do **not** work here.
> For the fully interactive platform, run the real app:
>
> ```bash
> python3.12 run.py    # then open http://localhost:8000
> ```

Snapshots included: landing, login, register, course catalogue, course detail, lesson
viewer, labs list, lab challenge, student dashboard, certificates list, printable
certificate, public verification, admin overview, admin course manager.
