Web Application Security: OWASP Top 10

Hack the web, then fix it

The OWASP Top 10 is the industry-standard list of the most critical web application security risks. In this track you will understand each risk, see how it is exploited, and learn how to remediate it.

Highlights

  • - Injection (SQLi) and how parameterized queries stop it
  • - Broken access control (IDOR, privilege escalation)
  • - Cross-site scripting (XSS) and output encoding
  • - Security misconfiguration and secure headers
